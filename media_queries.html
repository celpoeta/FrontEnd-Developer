Ejemplos de Media Queries

1. Media query simple: Cambia de color si el ancho es menor o igual a 600px.
2. Media query NOT: El texto cambia de color si el ancho es menor a 800px.
3. Media query ONLY: Aplicado solo en pantallas modernas con menos de 768px.
4. Media query AND: Este bloque desaparece entre 600px y 1024px.
